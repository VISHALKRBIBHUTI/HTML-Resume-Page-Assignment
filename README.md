# HTML Resume Template

A simple and clean resume template built using HTML.

## Demo
Open `Resume.html` in a browser to view the resume.

## Features
- Simple and clean layout
- Easy to customize personal details
- Beginner-friendly HTML structure
- Lightweight and fast loading
